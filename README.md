# Backbone.js Veterinary clinics list project

A simple Backbone.js project for listing and detailing Veterinary Clinics.

## Installation

in project folder using bower:

`bower install`

then, install globally using npm:

`npm i json-server`

## Running

In project folder:

`json-server --watch data/db.json`

after that, just open `index.html` file in your browser
